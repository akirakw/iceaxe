# IceAxe - Java High-level-API

## Requirements

* Java `>= 11`

* access to installed dependent modules:
  * tsubakuro

## How to build

```bash
cd tsubakuro
./gradlew publishMavenJavaPublicationToMavenLocal

cd iceaxe
./gradlew build
```
